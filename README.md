# Pageshelf

A free and open-source Pages server, written in Rust.

## Supported software

- [x] Forgejo

## Features

- [x] Dynamic hosting of sites
- [x] Integration Tested
  - [x] In-Memory Mock
- [x] Redis Support
- [ ] Prometheus endpoint
- [ ] 100% customizable UI
- [ ] Auth-locking of sites
