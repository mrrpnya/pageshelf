# Pageshelf

A free and open-source Pages server, written in Rust.

***Not ready for use yet.***

## Supported software

- [x] Forgejo

## Features

- [x] Dynamic hosting of sites
- [ ] Prometheus endpoint
- [ ] 100% customizable UI
- [ ] Auth-locking of sites
